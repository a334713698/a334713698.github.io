题目：有效的括号
解题思路：栈
示例：

输入： “( ) [ ] { }”
输出： true

输入： “( [ ) ]”
输出： false

代码：

算法理解：
当我们在遍历的时候，遇到左括号，就希望在后续中有右括号与之对应。根据先出现的左括号，延后对应，后出现的左括号，遇到右括号先对应的特点，栈的特点非常适合这道题的解题思路（先进后出）。
我们可以依次将一个符号压入栈顶，当有左括号遇到右括号时，两两相消，一直到最后如果栈内符号全部消除，则说明这组符号是有效的。
